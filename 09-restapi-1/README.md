
# Rest API Exercise 1

This folder contains the code for the exercise one of the Lemoncode REST API module.

This is based on a scaffolding and we need to implement two features, Character-Collection and Character using [rick and morty api](https://rickandmortyapi.com/)

# Setup

Open the code using the Reopen in Container feature from Dev Containers extension in VS Code.

# Install dependencies

To install dependencies execute the following command:

```bash
npm install
```


# Start

To start the exercise

```bash
npm start:dev
```
